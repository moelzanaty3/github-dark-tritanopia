# Changelog

All notable changes to the "GitHub Dark Tritanopia" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.3] - 2026-01-17

### Added
- Support for Cursor AI editor
- Enhanced markdown syntax highlighting
- Improved UI consistency across all components
- Professional documentation and contribution guidelines
- Extended language support enhancements

### Changed
- Refined color palette for better accessibility
- Updated theme description and categorization
- Improved visual hierarchy in editor UI elements

### Fixed
- Minor color consistency issues
- Enhanced contrast ratios for better readability

## [0.0.1] - 2026-01-16

### Added
- Initial release of GitHub Dark Tritanopia theme
- Complete UI color definitions (150+ colors)
- Comprehensive syntax highlighting (250+ token colors)
- Support for 40+ programming languages
- Optimized colors for tritanopia color vision deficiency
- Full VS Code compatibility

### Features
- Dark theme based on GitHub's design system
- Accessibility-first color selection
- Professional syntax highlighting
- Support for popular frameworks (React, Vue, etc.)
- Terminal color definitions
